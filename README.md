# FanDuel (fanduel)

FanDuel is a major US sports betting and daily fantasy sports operator (FanDuel Sportsbook, FanDuel Casino, FanDuel Fantasy, FanDuel TV). FanDuel does not publish a public developer REST API for its sportsbook, casino, or fantasy products. Internal/private APIs power the apps and a Sportradar/SBTech-derived trading stack but are not available to third-party developers.

## APIs

FanDuel does not publish a public developer REST API for its Sportsbook, Casino, Racing, or Fantasy products. Integration is limited to the FanDuel Affiliates / Partners program (marketing and lead-gen, not data access). Sports data and odds for third-party developers are typically licensed via Sportradar, Genius Sports, or other data vendors — not directly from FanDuel.
